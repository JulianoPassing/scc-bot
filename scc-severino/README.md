# scc-severino

Bot unificado do SCC, contendo todas as funções dos bots antigos:
- scc-avaliacoes
- scc-boost
- scc-sugestoes
- scc-ticket
- scc-ticket-s-wl
- scc-wl

## Como usar
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Copie o arquivo `.env.example` para `.env` e coloque seu token do Discord.
3. Inicie o bot:
   ```bash
   npm start
   ```

Cada módulo está em `modules/NOME_DO_MODULO`.
